typedef struct CPDMA_PACKET_ CPDMA_Packet

CPPI Packet.

The following is the format of a single packet to be transmitted via the CPDMA. For the CPDMA, a packets consists of one or more buffer descriptors, linked in a chain (or queue). The first buffer descriptor is the Start-Of-Packet (SOP) descriptor, and the last buffer descriptor is the End-Of-Packet descriptor.

typedef struct CPDMA_PACKET_QUEUE_ CPDMA_PacketQueue

Packet Queue.

A queue of packets, which are use for managing the active queue of of packets given to the hardware driver. Can alos be used by the translation/application layer to manage packets that are not currently active.
